# git + ipfs 파일관리전략

`git`으로 관리하기 어려운 대용량 파일에 대해서 LFS대신 `ipfs`로 공유하거나 내려받습니다.

따라서 이 도구를 사용하기 위해선, 위의 두 명령줄 도구가 필요합니다.
